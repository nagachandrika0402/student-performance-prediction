## 🚀 Project Overview
# Student Performance Prediction

This project predicts student academic performance based on study hours and sleep patterns using a Machine Learning model.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Description
The dataset is created and processed using Pandas. A Linear Regression model is trained to predict student marks based on input features such as study hours and sleep duration.

The project demonstrates fundamental concepts of machine learning including data preprocessing, model training, and prediction.

## Features
- Data preprocessing and analysis
- Linear Regression model implementation
- Prediction of student marks
- Data visualization using graphs

## Output
